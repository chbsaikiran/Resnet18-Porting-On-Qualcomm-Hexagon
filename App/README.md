# App — CIFAR-10 Web Classifier

A Flask web app that lets you upload an image and classifies it using the
exported ResNet18 ONNX model. Preprocessing matches the training pipeline
(OpenCV resize to 32x32, CIFAR-10 mean/std normalisation).

## Setup

```bash
cd App
pip install -r requirements.txt
```

## Run

Make sure the ONNX model exists at `Training/resnet18_cifar10.onnx`
(generated by `python onnx_dump.py` in the Training folder), then:

```bash
python app.py
```

Open **http://localhost:5000** in your browser.

### Custom model path

```bash
ONNX_MODEL=/path/to/model.onnx python app.py
```

## Usage

1. Click the upload area (or drag & drop) to select an image
2. Press **Classify**
3. The top-5 predicted CIFAR-10 classes are shown with confidence bars

## CIFAR-10 Classes

airplane, automobile, bird, cat, deer, dog, frog, horse, ship, truck
